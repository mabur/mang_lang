Built-in Functions

The following functions are built-in, which means that their implementations are not written in Manglang.

List Functions

new
Takes either a list or a string and returns an empty container of the same type.
prepend
prepend!(item list) returns a list with item added to the beginning of the list.

Boolean Functions

boolean
Convert a value to a boolean. All values are converted to yes, except empty list () and empty string "" and number zero 0 and the boolean no which are all converted to no.
not
Convert a value to a boolean. All values are converted to no, except empty list () and empty string "" and number zero 0 and the boolean no which are all converted to yes.
all
Takes a list of booleans and returns yes if all elements are yes and otherwise no.
any
Takes a list of booleans and returns yes if any element is yes and otherwise no.
none
Takes a list of booleans and returns yes if all element are no and otherwise yes.
equal
Takes two values and returns yes if they are equal and otherwise no.
unequal
Takes two values and returns no if they are equal and otherwise yes.

Functions on List of Numbers

add
Add a list of numbers.
mul
Multiply a list of numbers.
sub
Subtract two numbers.
div
Divide two numbers.
min
Minimum of a list of numbers.
max
Maximum of a list of numbers
less
Takes a list of numbers and returns yes if they are strictly increasing, and otherwise no
less_or_equal
Takes a list of numbers and returns yes if they are increasing, and otherwise no.

Functions on Number

round
Round a number to closest integer.
round_up
Round a number up to closest integer.
round_down
Round a number down to closest integer.
abs
Absolute value of a number.
sqrt
Square root of a number.

Character Functions

number
Takes a character and returns its ascii number.
character
Takes a number and returns the character with that ascii number.

Functions in Standard Library

The following functions are implemented in the standard library, which means that their implementation are written in Manglang.

Number functions

inc
Add 1 to a number.
dec
Subtract 1 from a number.

Character Functions

is_digit
Is a character a digit? Returns yes or no.
is_letter
Is a character a letter? Returns yes or no.
is_upper
Is a character upper case? Returns yes or no.
is_lower
Is a character lower case? Returns yes or no.
to_upper
Takes a character and converts it to upper case. Only has an effect it the character is a lower case letter.
to_lower
Takes a character and converts it to lower case. Only has an effect it the character is an upper case letter.

Container Functions

map
map!(f container) returns a list with the function f applied to each element in the container.
map_string
map_string!(f container) returns a string with the function f applied to each element in the container.
range
range!n returns a list with n elements.
enumerate
enumerate!containers returns a list like ({index=0 item=...} {index=1 item=...} ...)

Generic Container Functions

These functions work on both lists and strings, and the type of the output container is the same as the input.
drop
drop!(n container) returns a container with the first n elements removed.
take
take!(n container) returns a containers with the first n elements.
get_index
get_index!(n container) returns the nth element from the container.
reverse
reverse!container returns a container with the elements in the reverse order.
concat
concat!(a b) concatenates two containers.
clear_item
clear_item!(item container) returns a container with all occurances of the specified item removed.
clear_if
clear_if!(predicate container) returns a container without the items for which the predicate says yes.
replace
replace!(new_item container) returns a container where each element is replaced with new_item.
replace_item
replace_item!(old_item new_item container) returns a container with occurances of old_item replaced with new_item.
replace_if
replace_if!(predicate new_item container) returns a container where each element is replaced if the predicate says yes.
count
count!container returns the number of elements in the container.
count_item
count!(item container) counts the number of occurances of an item in the container.
count_if
count_if!(predicate container) counts the number of elements for which the predicate says yes.
